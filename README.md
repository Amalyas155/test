# Image Classification using Teachable Machine

## Project Description
This project is an image classification model created using Google Teachable Machine. The model was trained to recognize two classes: **Cats** and **Birds**.

## Tools Used
- Teachable Machine
- Google Colab
- Python

## Steps
1. Collected images of cats and birds.
2. Uploaded the images to Google Teachable Machine.
3. Trained the image classification model.
4. Exported the model.
5. Copied the generated code to Google Colab.
6. Tested the model using a new image.

## Result
The model successfully classified the test image with a confidence score of **99.998%**.
